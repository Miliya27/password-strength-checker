 **Interactive Password Checker**

A lightweight, client-side password strength validator. This project demonstrates DOM manipulation, Regular Expressions (Regex), and real-time UI updates.

## **Features**
- **Real-time Feedback:** Updates as the user types.
- **Visual Strength Meter:** Color-coded bar (Red -> Orange -> Green).
- **Security-First:** No backend involved; passwords never leave the user's browser.
- **Show/Hide Toggle:** Improves user experience.

##  Built With
- HTML5
- CSS3 (Flexbox & Transitions)
- Vanilla JavaScript (ES6+)
